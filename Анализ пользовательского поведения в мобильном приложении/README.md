# Анализ пользовательского поведения в мобильном приложении

**Сфера деятельности**:   
Стартапы, Бизнес, Интернет-сервисы
 
**Навыки и инструменты**:   
A/B-тестирование, Python, Pandas, Matplotlib, Seaborn, событийная аналитика, продуктовые метрики, Plotly, проверка статистических гип

**Задачи проекта**:   
На основе данных использования мобильного приложения для продажи продуктов питания проанализировать воронку продаж, а также оценить результаты A/A/B-тестирования.

**Описание проекта и выводы**:   
В данном проекте мной были изучены принципы событийной аналитики. Построена воронка продаж, исследован путь пользователей до покупки. По результату анализа воронки событий можно отметить следующие моменты: От первого экрана до оплаты "доходит" 47.70% пользователей. Точки потенциального роста продаж - улучшение конверсии в шаг с экрана оффера в корзину - сейчас почти 20% пользователей уходит. А также из корзины к оплате - сейчас на этом шаге теряется 14% пользователей.

В рамках исследования также были проанализированы результаты A/A/B-эксперимента с группами пользователей, которым демонстировался старый шрифт и экспериментальная группа с новым шрифтом. По результатам теста можно сказать, что изменение шрифта не сказалось на изменение поведения пользователей.

**Статус проекта**:  
Окончен
